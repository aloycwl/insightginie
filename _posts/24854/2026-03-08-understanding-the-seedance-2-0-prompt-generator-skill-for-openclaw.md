What is the Seedance 2.0 Prompt Generator Skill?
------------------------------------------------

The Seedance 2.0 Prompt Generator skill is a specialized tool designed to create cinematic video prompts for ByteDance’s Seedance 2.0 AI video model. This skill activates when users request video prompts, Seedance prompts, AI video generation prompts, or anything related to Seedance 2.0 prompt crafting.

Core Architecture of Seedance Prompts
-------------------------------------

Every prompt generated by this skill follows a strict architectural order to ensure optimal results. The skill emphasizes that deviating from this structure can cause “drift” in the generated content. The six key components are:

1. **Subject** – The primary character or element with specific details
2. **Action** – What happens, described in present tense verb phrases
3. **Camera** – How we see the action, including shot size and movement
4. **Style** – The visual aesthetic, lighting, and color treatment
5. **Audio** – What we hear, including music and sound effects
6. **Constraints** – Guardrails and limitations for the generation

### Subject Details

The skill emphasizes focusing on one primary subject to avoid splitting the model’s attention. For each subject, it recommends including specific details like age, material, clothing, and distinguishing features. For example: “Wooden Koda creature with glowing orange eyes, green gems on head, purple cape.”

### Action Sequencing

Actions should be described using specific verb phrases in present tense. For complex sequences, the skill recommends breaking down the action “beat by beat” and chaining these beats chronologically. Each beat should contain one action, with multiple beats separated by commas.

### Camera Specifications

The skill provides detailed guidance on camera work, emphasizing that shot size should come first, followed by movement, angle, and lens feel. It includes a comprehensive shot cheat sheet showing which camera techniques pair best with different shot types. For compound camera moves, the skill recommends separating them into distinct beats.

Advanced Prompt Templates
-------------------------

The skill includes several pre-built templates for different video scenarios:

* **Cinematic Scene** – For general atmospheric videos
* **Multi-Shot Narrative** – For videos with multiple distinct shots
* **Action Sequence** – For high-energy, dynamic content

Reference System and Multi-Reference Support
--------------------------------------------

One of the skill’s powerful features is its reference system using @tags. Users can upload images, videos, and audio that the skill can reference in prompts:

* @Image1, @Image2, etc. for uploaded images
* @Video1, @Video2, etc. for uploaded videos
* @Audio1, etc. for uploaded audio

The skill supports up to 9 images, 3 videos, and 3 audio clips per generation, with various usage patterns including character identity, first/last frame specification, motion transfer, and style reference.

Negative Prompt Checklist
-------------------------

To ensure clean, professional results, the skill includes a comprehensive negative prompt checklist. Users are advised to select 3-5 items from categories including:

* Visual noise (no text overlays, watermarks, etc.)
* Identity drift (no extra characters, crowd, etc.)
* Camera chaos (no snap zooms, whip pans, etc.)
* Body artifacts (no extra fingers, deformed hands, etc.)
* Branding (no logos, labels, etc.)
* Color issues (no neon lighting, heavy teal/orange, etc.)
* Environment problems (no unwanted rain/fog/smoke, etc.)

Clean High-Motion Technique
---------------------------

The skill incorporates advanced techniques learned from real-world results to produce sharp, blur-free motion even at extreme speeds. These include:

* **The Continuous Shot Lock** – Declaring “single continuous shot” upfront for temporal coherence
* **Physics-Motivated Camera** – Every camera move needs a physical verb with clear motivation
* **Environmental Anchoring** – Using static reference geometry for stable backgrounds
* **Scale Progression Arc** – Structuring as Macro → Micro → Macro for clear resolution targets
* **Sensory Render Instructions** – Using computable effects instead of mood words
* **Rhythm Through Verbs** – Pacing through action chain length rather than timers

Practical Application
---------------------

This skill is particularly useful for content creators, filmmakers, and marketers who need to generate specific video content using Seedance 2.0. By following the skill’s structured approach, users can create prompts that produce consistent, high-quality results tailored to their creative vision.

The skill essentially serves as a bridge between creative intent and technical execution, translating artistic concepts into the precise language that Seedance 2.0 understands best. This makes it an invaluable tool for anyone looking to leverage AI video generation effectively.
