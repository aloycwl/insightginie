What is the OpenDART Disclosure Skill?
--------------------------------------

The OpenDART Disclosure skill is an advanced AI-powered tool designed to access, analyze, and summarize corporate disclosures from Korea’s Electronic Disclosure System (DART). This skill leverages the OpenDART API to provide users with real-time access to financial reports, regulatory filings, and corporate announcements from Korean companies.

OpenDART (Open Data Analysis & Retrieval of Transparency) is the official public data portal operated by the Financial Supervisory Service of Korea. It provides comprehensive access to corporate disclosures, financial statements, and regulatory reports filed by companies listed on Korean stock exchanges.

---------------------------------------

The skill operates through a streamlined workflow that combines AI capabilities with direct API integration. Here’s a breakdown of its functionality:

### Input Collection and Processing The skill begins by collecting essential inputs from users: * **Company Identifier**: Users can provide either a corp\_code (preferred) or company name * **Date Range**: Specified using bgn\_de and end\_de parameters in YYYYMMDD format * **Filing Type Filters**: Optional filters for regular reports (A), major issues (B), shares (C), etc. * **Desired Output**: Users can request latest N items, only links, or short summaries Company Identity Resolution The skill employs a sophisticated company identification process: 1. If users provide a corp\_code, it’s used directly for API queries 2. If only a company name is given, the skill runs the search-corp script to find matching companies 3. The system handles fuzzy matching and can show multiple candidates if several companies have similar names Data Retrieval and Processing Once the company is identified, the skill: 1. Pulls disclosures using the recent command 2. Sorts results by latest filing date 3. Applies user-specified filters and limits 4. Extracts key information including filing date, report name, receipt number, and OpenDART link AI-Powered Analysis and Summarization The skill’s AI capabilities shine in its ability to: * Generate
