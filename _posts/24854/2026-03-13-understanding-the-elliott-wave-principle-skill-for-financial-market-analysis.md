The Elliott Wave Principle skill provides comprehensive guidance for applying this powerful technical analysis method to financial market price data. Core Purpose of the Elliott Wave Principle Skill
------------------------------------------------

The primary function of this skill is to assist users in applying the Elliott Wave Principle to analyze financial market price data. It contains reference files with detailed rules, pattern descriptions, and practical guidelines that enable systematic market analysis. When users ask about wave counting, identifying impulse waves, corrective patterns, Fibonacci retracements, or any Elliott Wave concepts, this skill provides expert guidance based on established principles.

Quick Start for Market Analysis
-------------------------------

When users provide chart data or price information and request wave analysis, the skill follows a systematic approach. First, it determines the trend direction and looks for the basic 5-wave pattern (impulse) followed by a 3-wave correction. The skill then uses reference files to verify rules and identify specific patterns, ensuring accurate analysis based on established Elliott Wave methodology.

Comprehensive Reference Library
-------------------------------

The skill includes modular reference files stored in the references/ directory, allowing users to load specific information based on their needs. These files include basic-principles.md covering fundamental concepts like the 5-3 wave structure and wave degrees, impulse-waves.md detailing rules for impulse waves and extensions, and corrective-patterns.md describing zigzags, flats, triangles, and combinations.

Additional reference files include guidelines.md for practical application tips like alternation and channeling, fibonacci-relationships.md for understanding Fibonacci ratios in wave relationships, and glossary.md for definitions of Elliott Wave terms. This modular design allows users to load only the necessary references, keeping analysis efficient and focused.

Practical Application Methodology
---------------------------------

The skill follows a structured approach to market analysis. First, it understands the user’s request to determine if they need wave counting, pattern identification, or concept explanation. Then, it loads the relevant reference files based on the specific question. For example, if a user asks about a particular corrective pattern, the skill loads corrective-patterns.md, while questions about wave counting rules trigger loading of impulse-waves.md and basic-principles.md.

The skill applies established rules to analyze provided price data, remembering that the Wave Principle is probabilistic and multiple interpretations may be valid. This transparent approach helps users understand the methodology behind each analysis.

Critical Rules and Guidelines
-----------------------------

The skill prioritizes established rules over guidelines when analyzing market data. Key rules include that wave 3 cannot be the shortest impulse wave, wave 4 cannot overlap wave 1 in an impulse pattern, and wave 2 cannot retrace more than 100% of wave 1. These hard rules take precedence over softer guidelines like alternation or channeling.

When uncertainty arises, the skill consults the glossary for term definitions to ensure accurate understanding. The skill also recognizes that while examples in reference files illustrate typical patterns, actual market patterns may vary due to the dynamic nature of financial markets. For comprehensive analysis, the skill can load multiple references simultaneously, though it’s designed to be modular for efficiency.
