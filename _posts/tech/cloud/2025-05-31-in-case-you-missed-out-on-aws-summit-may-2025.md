Had an amazing time at **AWS Summit 2025** at Marina Bay Sands on 29 May! It was a high-energy day packed with **learning, innovation, and networking**.

AWS once again showcased why they're the leader in **cloud computing** and **AI as a Service**, offering seamless solutions that connect **developers, vendors**, and **end users**. From infrastructure to innovation, there's almost nothing you *can't* build on AWS — and do it with ease.

The event was full of surprises — including a giant slide that took you from the 5th to the 4th floor in seconds (yes, with style!). Despite a slight delay at registration, the experience was incredibly well-organized and immersive.

While most of the content focused on AI and real-world use cases, Web3 was still underrepresented. I'll be sharing some of the highlights soon, including:

* Vendor showcases and how they're using AWS in practical ways
* Key AWS services discussed
* Insights into adoption strategies across industries

If you're a startup or builder, don't forget to check out **[AWS Activate](https://aws.amazon.com/activate/)** — a great way to get **free AWS credits** to kickstart your journey.

Key AWS Showcase
----------------

### GetGo CI/CD example

![](https://insightginie.com/wp-content/uploads/2025/05/2505311050.avif)

* **AWS CodePipeline**: Automates the build, test, and deploy phases of your release process.
* **AWS CodeBuild**: Compiles source code, runs tests, and produces software packages.
* **AWS S3**: Scalable object storage for data backup, archiving, and content delivery.

#### **Related mentions**

* **Amazon Q Developer:** AI-powered assistant that helps developers write, debug, and understand code faster.
* **Amazon Nova:** Internal Amazon service for deploying software safely and reliably at scale.
* **AWS Amplify:** Streamlines building, deploying, and hosting full-stack web and mobile applications.

### Authentication example

![](https://insightginie.com/wp-content/uploads/2025/05/2505311057.avif)

* **AWS Lambda JWT**: Validates JSON Web Tokens in serverless functions for secure, stateless authentication.
* **Amazon API Gateway**: Creates, publishes, and secures APIs at scale with integrated monitoring and throttling.
* **AWS ElastiCache**: In-memory data store service for caching and real-time applications.
* **AWS PrivateLink**: Enables private connectivity to AWS services and VPC endpoints without exposing traffic to the internet.
* **Amazon Cloud Map**: Service discovery for cloud resources with dynamic DNS and health checking.
* **AWS Fargate**: Serverless compute engine to run containers without managing servers.
* **AWS Aurora**: High-performance, fully managed relational database compatible with MySQL and PostgreSQL.

#### **Related mentions**

* **AWS Cognito:** Provides user sign-up, sign-in, and access control for web and mobile apps.
* **AWS AppSync:** Managed service for building GraphQL APIs with real-time data and offline capabilities.

### Other notable services

#### AI related

* **Amazon Bedrock**: Enables building and scaling generative AI apps using foundation models via an API.
* **Amazon QuickSight**: Cloud-powered business intelligence service for creating interactive dashboards and visualizations.
* **Amazon SageMaker**: Fully managed service to build, train, and deploy machine learning models at scale.

#### Gaming related

* **AWS Card Clash**: A sample game developed by AWS to demonstrate scalable backend architecture for multiplayer card games.
* **AWS Gaming Marketplace**: A curated catalog of game development tools and services available for purchase or deployment on AWS.

#### Administrative related

* **AWS Config**: Tracks and audits AWS resource configurations and changes over time.
* **AWS Control Tower**: Automates setup and governance of a secure, multi-account AWS environment.
* **AWS CloudWatch**: Monitors AWS resources and applications with logs, metrics, and alarms.

**Bonus Insight from the Asia Blockchain Association Side Event**
-----------------------------------------------------------------

Exciting discussions around **High-Performance Computing (HPC)** and its role in the future of Web3!

* Projects like **Golem Chain** are pioneering decentralized HPC, creating alternatives to traditional cloud infrastructures.
* **HPC is poised to enhance quantum-proofing** of blockchain systems — a critical step toward future-proof Web3 security.
* By **2030**, Web3 apps may no longer rely on centralized cloud providers. Instead, they'll thrive in **decentralized HPC marketplaces**, offering better speed, privacy, and cost-efficiency.
* Web3 + HPC = The next evolution in decentralized computing.
