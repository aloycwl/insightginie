<h2>TokenWatch: Track and Optimize Your AI Token Usage and Costs</h2>
<p>TokenWatch is a free and open-source skill designed to help you track, analyze, and optimize token usage and costs across AI providers. It allows you to set budgets, receive alerts, compare models, and reduce your overall spend.</p>
<h3>Why This Skill?</h3>
<p>After OpenAI&#8217;s acquisition of OpenClaw, token costs have become the #1 concern for power users. This skill gives you full visibility into what you&#8217;re spending, where it&#8217;s going, and exactly how to reduce it.</p>
<p>Problems it solves:</p>
<ul>
<li>You don&#8217;t know how much you&#8217;re spending until the bill arrives</li>
<li>No way to compare costs across providers before choosing a model</li>
<li>No alerts when you&#8217;re approaching your budget</li>
<li>No actionable suggestions for reducing spend</li>
</ul>
<h3>Features</h3>
<h4>1. Record Usage &#038; Auto-Calculate Costs</h4>
<p>You can manually record API calls and automatically calculate their costs:</p>
<pre><code>from tokenwatch import TokenWatch
monitor = TokenWatch()
monitor.record_usage(
    model="claude-haiku-4-5-20251001",
    input_tokens=1200,
    output_tokens=400,
    task_label="summarize article"
)
# ✅ Recorded: $0.00192
</code></pre>
<h4>2. Auto-Record from API Responses</h4>
<p>TokenWatch can automatically extract token counts from API responses:</p>
<pre><code>from tokenwatch import record_from_anthropic_response, record_from_openai_response

# Anthropic
response = client.messages.create(
    model="claude-haiku-4-5-20251001"
    ...
)
record_from_anthropic_response(monitor, response, task_label="my task")

# OpenAI
response = client.chat.completions.create(
    model="gpt-4o-mini"
    ...
)
record_from_openai_response(monitor, response, task_label="my task")
</code></pre>
<h4>3. Set Budgets with Alerts</h4>
<p>Configure spending limits and receive automatic alerts:</p>
<pre><code>monitor.set_budget(
    daily_usd=1.00,
    weekly_usd=5.00,
    monthly_usd=15.00,
    per_call_usd=0.10,
    alert_at_percent=80.0
)
# ✅ Budget set: daily=$1.0, weekly=$5.0, monthly=$15.0
# 🚨 BUDGET ALERT fires automatically when threshold is crossed
</code></pre>
