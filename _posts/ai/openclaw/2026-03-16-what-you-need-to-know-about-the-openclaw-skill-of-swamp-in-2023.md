<h2>Swamp Workflow and Commands</h2>
<p>Swamp uses several workflows and commands to help you bring your API models to life. Let’s look into a few key commands:</p>
<ul>
<li><code>swamp repo init --tool claude</code>: Before any operation, Swamp sets up a repository with necessary structures. For Claude/OpenClaw AI integration, the <code>--tool claude</code> flag is used.</li>
<li><code>swamp model create &lt;type&gt; &lt;name&gt;</code>: Creates a new model definition. For example, <code>swamp model create command/shell github-issues</code> would set up a model for GitHub issues.</li>
<li><code>swamp model method run &lt;model_name&gt; &lt;method_name&gt;</code>: This command executes a specific method on your model, helping you test it properly.</li>
</ul>
