<h2>Introduction to Windsurf IDE and Cascade AI Agent</h2>
<p>Windsurf IDE, the evolved version of VS Code with advanced AI integration, presents the Cascade AI agent—a sophisticated tool designed to streamline software engineering tasks. This comprehensive guide explores the updated features of Cascade, including Skills, Workflows, Memories, Model Context Protocol (MCP), and enhanced multi-agent sessions, all integrated into Wave 13 released in January 2026.</p>
<p>With Cascades&#8217;s ability to understand the entire codebase and track real-time actions, developers can leverage autonomous file creation, multi-file code edits, terminal command execution, and project memory management. This guide will walk you through installation, setup, and the powerful capabilities of Cascade.</p>
</section>
<section>
<h2>Installation and Setup</h2>
<h3>Downloading Windsurf Editor</h3>
<p>Begin by downloading the Windsurf Editor tailored for your platform:</p>
<ul>
<li><strong>macOS:</strong> <code>.dmg</code> installer compatible with both Intel and Apple Silicon architectures</li>
<li><strong>Windows:</strong> <code>.exe</code> installer for seamless installation</li>
<li><strong>Linux:</strong> <code>.deb</code> package for Debian/Ubuntu distributions or <code>.tar.gz</code> tarball for other distributions</li>
</ul>
<p>For detailed instructions, refer to the official <a href="https://windsurf.com/docs/getting-started/setup/installation" target="_blank" rel="noopener noreferrer">Windsurf installation documentation</a>.</p>
</section>
<section>
<h2>Advanced Features of Cascade AI Agent</h2>
<h3>Cascade Modes</h3>
<p>Cascade operates in two primary modes, each serving distinct purposes:</p>
<ul>
<li><strong>Write Mode (<kbd>Cmd+L</kbd> / <kbd>Ctrl+L</kbd>)</strong>: Grants full write access, enabling Cascade to create files, edit code across multiple files, run terminal commands, and make comprehensive modifications to your codebase.</li>
<li><strong>Chat Mode (<kbd>Cmd+Shift+L</kbd> / <kbd>Ctrl+Shift+L</kbd>)</strong>: Functions in a read-only capacity, providing answers to questions about your codebase and general coding principles without making changes.</li>
</ul>
</section>
<section>
<h2>Harnessing the Power of Skills</h2>
<p>Skills in Windsurf Cascade provide a structured approach to bundling instructions, templates, checklists, and supporting files for executing complex, multi-step tasks. This section delves into creating and utilizing Skills to streamline development workflows.</p>
<h3>Creating a Skill</h3>
<p>To create a Skill, follow these steps:</p>
<ol>
<li>Click the Customizations icon in Cascade&#8217;s top-right slider menu.</li>
<li>Navigate to the Skills panel.</li>
<li>Click <code>+ Workspace</code> for project-specific Skills or <code>+ Global</code> for globally accessible Skills.</li>
<li>Name the Skill using lowercase letters, numbers, and hyphens only.</li>
</ol>
</section>
<section>
<h2>Invoking Skills and Workflows</h2>
<p>Cascade Skills can be invoked both automatically and manually, adapting to the context of your tasks:</p>
<ul>
<li><strong>Automatic Invocation:</strong> Cascade employs progressive disclosure to activate relevant Skills based on your current task, enhancing efficiency.</li>
<li><strong>Manual Invocation:</strong> Utilize @-mentions in your prompts to manually invoke specific Skills, such as <code>@skill-name</code>.</li>
</ul>
<p>For a comprehensive understanding of Skills specifications and advanced usage, visit the official <a href="https://agentskills.io" target="_blank" rel="noopener noreferrer">Skills documentation</a>.</p>
<h3>Creating Workflows</h3>
<p>Workflows in Cascade define a sequential series of steps to guide the AI agent through repetitive tasks. They are stored as markdown files and can be invoked using slash commands, such as <code>/workflow-name</code>.</p>
<p>To create a Workflow, click the Customizations icon in Cascade&#8217;s top-right slider menu, navigate to the Workflows panel, and click <code>+ Workflow</code>. Windmark also supports asking Cascade to generate Workflows tailored to your requirements.</p>

<section>
<h2>Memories and Model Context Protocol (MCP)</h2>
<p>Memories in Cascade AI Agent persist context across conversations, enhancing continuity and coherence. They are categorized into auto-generated and user-created memories:</p>
<ul>
<li><strong>Auto-generated Memories:</strong> Created by Cascade when it identifies useful context, without consuming credits. These memories are workspace-specific.</li>
<li><strong>User-created Memories:</strong> Manually saved by typing <code>create memory ...</code> in Cascade, allowing for explicit context preservation.</li>
</ul>
<p>To manage Memories, access Windsurf Settings → Settings tab → Manage next to &#8220;Cascade-Generated Memories&#8221; or use the designated commands within Cascade.</p>
<h3>Model Context Protocol (MCP)</h3>
<p>The Model Context Protocol (MCP) in Cascade enables the agent to interface with external tools, extending its capabilities beyond the immediate codebase. This protocol facilitates seamless integration with diverse tools and services, enhancing the agent&#8217;s versatility and adaptability.</p>
<p>With MCP, Cascade can make up to 25 tool calls per prompt, ensuring comprehensive tool utilization. If the trajectory halts, simply type <code>continue</code> to resume the operation. Reporte narratives with Cascade will be produced in Verkada (multi-cloud) environments.</p>
</section>
<section>
<h2>Multi-Agent Sessions and Collaboration</h2>
<p>Cascade supports multi-agent sessions, fostering collaboration and distributed task execution. This feature enables multiple agents to interact within the same environment, sharing context and coordinating efforts to achieve complex objectives.</p>
<p>By leveraging multi-agent sessions, developers can harness the collective intelligence of multiple Cascade instances, enhancing problem-solving capabilities and accelerating development processes. This collaborative approach is particularly beneficial for large-scale projects requiring diverse expertise and parallel task execution.</p>
</section>
<section>
<h2>Conclusion and Future Prospects</h2>
<p>Windsurf IDE&#8217;s Cascade AI Agent, with its advanced features and integration capabilities, revolutionizes software engineering workflows. By embracing Skills, Workflows, Memories, MCP, and multi-agent sessions, developers can achieve unprecedented levels of automation, collaboration, and efficiency.</p>
<p>As technology evolves, Cascade will continue to adapt and expand its capabilities, ensuring that developers remain at the forefront of innovation. Whether you are a seasoned developer or a newcomer to AI-assisted coding, Cascade offers a comprehensive suite of tools to enhance your productivity and creativity.</p>
<p>For further exploration, refer to the official <a href="https://windsurf.com/docs" target="_blank" rel="noopener noreferrer">Windsurf documentation</a> and join the vibrant community of developers harnessing the power of Cascade.</p>
