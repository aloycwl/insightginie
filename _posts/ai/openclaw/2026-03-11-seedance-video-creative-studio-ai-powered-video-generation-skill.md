<p>Introducing Seedance Video Creative Studio, an advanced AI skill that transforms your visual and textual inputs into compelling video content. This autonomous creative director analyzes images, expands copy, matches camera work, validates quality, and generates videos via API when users provide images and text.</p>
<h2>How Seedance Works</h2>
<p>Seedance operates as your personal video creative director. When you provide materials—whether images, copy, both, or even just an image with no words—it autonomously decides how to turn them into a creative, memorable Seedance video prompt, calling the API to generate when appropriate.</p>
<p>Unlike template-based systems, Seedance has no fixed process or mandatory step order. Your judgment IS the process. The system makes autonomous decisions about what to analyze first, whether to search for trending prompts, how many iterations to perform, and when to converge on a final output.</p>
<h3>Critical Language Requirement</h3>
<p>All prompts generated by Seedance must be in Chinese. While your conversation with the user can be in any language, the final video prompt output must always be Chinese, as Seedance understands Chinese best.</p>
<h2>Core Capabilities</h2>
<h3>Multimodal Vision</h3>
<p>Seedance directly analyzes images to understand scene composition, subject matter, shot scale, composition dynamics, color palette, and artistic style. This visual analysis forms the foundation for all creative decisions.</p>
<h3>Creative Ideation</h3>
<p>The system can diverge multiple creative directions from a single image, then pick the most interesting one. Rather than generating predictable outputs, Seedance seeks creative angles that will capture viewer attention.</p>
<h3>Copy Expansion</h3>
<p>Vague copy gets expanded into full prompts with detailed camera work, lighting specifications, rhythm descriptions, and stylistic elements. The system transforms simple concepts into comprehensive video directions.</p>
<h3>Web Search Integration</h3>
<p>Seedance can search trending prompt patterns and adapt phrasing into current copy styles, ensuring outputs remain contemporary and relevant to current creative trends.</p>
<h3>Vocabulary Selection</h3>
<p>The system pulls terms from reference.md cinematography and style vocabulary, never inventing terms. This ensures professional-quality output that aligns with industry standards.</p>
<h3>Image Diagnosis</h3>
<p>Seedance checks resolution (300–6000px), aspect ratio (0.4–2.5), and composition issues. It proactively flags camera risks or crop/adjust with Python preprocessing when needed.</p>
<h3>Pairing Validation</h3>
<p>The system judges whether image + prompt + camera work are harmonious, fixing mismatches locally to ensure cohesive final outputs.</p>
<h3>Creativity Review</h3>
<p>After writing a prompt, Seedance repeatedly asks itself &#8220;is this prompt interesting?&#8221; If not good enough, it scraps and redoes the work. Better to revise two extra rounds than output a mediocre prompt.</p>
<h2>Creativity Standards</h2>
<p>Seedance maintains high creative standards through a rigorous review process. After writing a prompt, the system doesn&#8217;t rush to generate. Instead, it passes the creativity gate first by asking critical questions:</p>
<ul>
<li>Is it memorable? What will the viewer remember after watching? If the answer is &#8220;nothing&#8221; — rewrite.</li>
<li>Is there surprise? All expected visuals = boring. A good prompt has at least one twist, contrast, exaggeration, or unusual detail.</li>
<li>Is there emotion? Purely descriptive visuals have no impact. Add emotional arcs: tension → release, calm → explosion, warmth → twist.</li>
<li>Is there narrative? Even in 5 seconds, there should be an A → B change, not a static showcase.</li>
</ul>
<p>If the prompt doesn&#8217;t meet these standards, Seedance iterates — changing angle, swapping style, adding conflict, restructuring narrative — until the system thinks &#8220;this is interesting.&#8221;</p>
<h2>Handling Image-Only Inputs</h2>
<p>When users drop an image without any accompanying copy, Seedance enters its most creative mode. The system:</p>
<ol>
<li>Reads the image: Analyzes scene, mood, story potential, and visual tension</li>
<li>Diverges creative directions: From the image, brainstorms 2–3 completely different angles</li>
<li>Picks the most interesting direction and develops it into a full prompt</li>
<li>Alternatively, presents directions for the user to choose</li>
</ol>
<p>For example, given a coffee cup photo, Seedance might generate:</p>
<ul>
<li>Healing: Steam rising from morning coffee slowly morphs into memory fragments</li>
<li>Commercial: Coffee beans fall from the sky, burst apart, and reassemble into a latte in 3D</li>
<li>Mystery: The patterns on the coffee surface slowly become a map, camera pushes in to enter another world</li>
</ul>
<p>Even in this mode, Seedance passes the creativity review when developing — &#8220;it runs&#8221; is not enough, it needs to be &#8220;interesting.&#8221;</p>
<h2>Quality Redlines</h2>
<p>Seedance maintains strict quality standards:</p>
<ul>
<li>Prompts must be in Chinese — ready to paste directly into Jimeng (即梦)</li>
<li>@ references use only @图片1~@图片9, @视频1~@视频3, @音频1~@音频3, each with purpose noted</li>
<li>Distinguish &#8220;reference&#8221; (borrow style/motion) from &#8220;edit&#8221; (modify the original)</li>
<li>No realistic human face materials</li>
<li>Camera/style terms from reference.md vocabulary only — never invent terms</li>
<li>Dialogue in quotes, tagged with character and emotion</li>
</ul>
<h2>Platform Specifications</h2>
<p>Seedance works within specific platform constraints:</p>
<ul>
<li>Images: jpeg/png/webp/bmp/tiff/gif, ≤9, each <30 MB</li>
<li>Videos: mp4/mov, ≤3, total 2–15s, each <50 MB</li>
<li>Audio: mp3/wav, ≤3, total ≤15s, each <15 MB</li>
<li>Mixed: Total ≤12 files</li>
<li>Output: 2.0: 4–15s; 1.x: 4‒12s; 2K resolution, native audio</li>
</ul>
<h2>API Generation</h2>
<p>The system uses scripts/seedance.py to call the Volcengine Ark API. Script defaults to Seedance 2.0, but falls back to —model doubao-seedance-1-5-pro-251215 if 2.0 API is unavailable or model errors occur.</p>

<h3>Usage Examples</h3>
<p>Text-to-video (2.0 default model):</p>
<pre><code>python3 scripts/seedance.py create --prompt "提示词" --ratio 16:9 --duration 5 --wait --download ~/Desktop
</code></pre>
<p>First frame image (use adaptive ratio with images):</p>
<pre><code>python3 scripts/seedance.py create --prompt "提示词" --image img.jpg --ratio adaptive --duration 5 --wait --download ~/Desktop
</code></pre>
<p>First + last frame:</p>
<pre><code>python3 scripts/seedance.py create --prompt "提示词" --image first.jpg --last-frame last.jpg --ratio adaptive --duration 5 --wait --download ~/Desktop
</code></pre>
<p>Video reference / motion replication (2.0):</p>
<pre><code>python3 scripts/seedance.py create --prompt "提示词" --video motion_ref.mp4 --wait --download ~/Desktop
</code></pre>
<p>Audio reference / beat-sync (2.0):</p>
<pre><code>python3 scripts/seedance.py create --prompt "提示词" --audio bgm.mp3 --wait --download ~/Desktop
</code></pre>
<p>Multimodal mix (image + video + audio, 2.0):</p>
<pre><code>python3 scripts/seedance.py create --prompt "提示词" --image img.jpg --video ref.mp4 --audio bgm.mp3 --ratio adaptive --wait --download ~/Desktop
</code></pre>
<p>Auto duration (model decides 4-15s, 1.5 Pro / 2.0):</p>
<pre><code>python3 scripts/seedance.py create --prompt "提示词" --duration -1 --wait --download ~/Desktop
</code></pre>
<p>Draft preview (low-cost, confirm then generate final, 1.5 Pro):</p>
<pre><code>python3 scripts/seedance.py create --prompt "提示词" --image img.jpg --draft
</code></pre>
