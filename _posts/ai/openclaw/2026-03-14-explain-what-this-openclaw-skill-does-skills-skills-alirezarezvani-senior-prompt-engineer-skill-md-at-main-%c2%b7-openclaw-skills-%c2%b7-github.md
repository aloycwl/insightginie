<p>The &#8216;senior-prompt-engineer&#8217; skill in the OpenClaw GitHub repository is designed to help users optimize, evaluate, and design prompts, as well as build agentic systems and implement RAG (Retrieval-Augmented Generation). This skill focuses on prompt engineering patterns, LLM evaluation frameworks, agent architectures, and structured output design.</p>
<h3>Key Features:</h3>
<ul>
<li><strong>Prompt Optimization:</strong> Users can analyze and optimize prompt files to improve token efficiency, clarity, and structure. The feature can generate optimized versions of prompts, count tokens for cost estimation, and extract and manage few-shot examples.</li>
<li><strong>RAG Evaluation:</strong> This skill includes a RAG Evaluator that measures the quality of retrieval-augmented generation by evaluating context relevance and answer faithfulness. Users can generate evaluation metrics and quality reports through this tool.</li>
<li><strong>Agent Orchestration:</strong> Users can parse agent definitions and visualize execution flows, as well as validate tool configurations using the Agent Orchestrator feature. This tool helps validate agent configurations, visualize workflows, and identify potential issues.</li>
<li><strong>Prompt Engineering Workflows:</strong> The skill also includes comprehensive workflows for prompt optimization and few-shot example design. These workflows are designed to help users improve their prompts, validate results, and create effective in-context learning examples.</li>
</ul>
<p>The skill is intended for use when users need to perform advanced prompt engineering tasks, evaluate LLM outputs, implement RAG, build agent-based systems, or design efficient AI workflows. By offering a range of tools and workflows, this skill enables more streamlined and effective prompt engineering processes.</p>
